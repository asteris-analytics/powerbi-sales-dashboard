# Europe Sales Dashboard

## Project Overview
This Power BI dashboard analyzes European sales performance using revenue, profit, countries, products and monthly trends.

## Tools Used
- Power BI
- Excel/CSV
- Data Visualization

## Key Insights
- Total Revenue exceeded 127B.
- Total Profit exceeded 39B.
- Cosmetics generated the highest total profit.
- Profit varied significantly across months.
- Several countries such as Andorra, Ukraine and Malta contributed strongly to total profit.
